### 'in' 연산자로 프로퍼티 존재 여부 확인하기
자바스크립트 객체는 존재하지 않는 프로퍼티에 접근하려 해도 에러가 발생하지 않고 `undefined`를 반환한다.
특징을 응용하면 프로퍼티 존재 여부를 쉽게 확인할 수 있다.

```javascript
"key" in object
```
