## 타이핑 공부법

> 백지 공부법과 유사하다. 핵심은 '인출'이다.

1. 15분 동안 책을 읽는다.
2. 책을 덮고, 배운 것을 쏟아낸다.
3. 부족한 부분을 다시 읽고 과정을 되풀이한다.
4. 과정을 반복한다.
