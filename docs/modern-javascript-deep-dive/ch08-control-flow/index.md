# 08. 제어문

## 8.1 블록문

```javascript
{
  var book = '유연함의 힘';
}
```

이전에 문(statement)를 작성할 때 세미콜론 사용을 권장했다. 하지만 블록문은 자체 종결성을 갖기 때문에 세미콜론을 사용하지 않는다.
