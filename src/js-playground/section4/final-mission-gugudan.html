학습 목표

  • 학습한 내용을 활용해 한 단계 더 난이도가 있는 문제를 구현한다.

🚀 최종 요구사항

  • 참고자료, 교재, 인터넷의 도움 없이 백지 상태에서 구구단을 구현
  • 사용자가 입력한 값에 따라 크기가 다른 구구단을 계산, 출력
  • 8, 7과 같은 문자열을 입력하면 팔칠단을 구현 ex) 2 * 1 ... 8 * 8
  • html, css, DOM 등을 배워서 화면을 구성

힌트

  • 사용자가 입력한 값을 문자열로 받음
  • 사용자가 입력한 값을 쉼표를 기준으로 분리
  • 쉼표를 기준으로 분리한 문자열을 숫자로 변환
  • 반복문을 실행해 곱셈 결과를 화면에 출력

⭐️ 조언

  • 과정 중 자신 없는 부분, 이해가 되지 않는 부분을 정리하고 해당 부분을 연습
  • 전체 과정을 반복 연습, 반복 연습하며 힌트를 줄여가며 연습
  • 각 단계에서 이해한 내용, 이해하지 못한 부분을 정리, 이해하지 못한 부분은 다른 미션을 통해 극복하기 위해 연습
  • 새롭게 등장하는 용어들을 자신의 이해 수준으로 정리하는 연습

🚨 왜?

  • ES6 문법을 어떻게 활용해야하는가? (반복문 개선)
  • 출력형식 지정 (html, css , DOM 활용)
  • 함수를 분리하여 구조화 하는 작업이 능숙하지 않음
